## 🌟 What is this?

This is the interactive portfolio of **Saurabh**, a passionate 3D Artist and Game Developer. Designed to be as immersive as the environments he creates, this web experience leverages cutting-edge frontend technologies to deliver smooth scrolling, physics-based interactions, and cinematic animations. 

Whether it's Unreal Engine, Blender, or Unity, Saurabh builds worlds. This website is the portal to them.

## 🛠️ The "Secret Sauce" (Tech Stack)

We didn't just build a website; we engineered a fluid experience. Here is the magic under the hood:

- ⚛️ **React 19 & Vite** - The engine. Blazing fast rendering and instantaneous development server.
- 🎨 **Tailwind CSS v4** - The paint. Rapid, utility-first styling for a sleek, modern, and responsive UI.
- 🎬 **GSAP & Framer Motion** - The choreographer. Handling complex timeline animations and dynamic component mount/unmount effects.
- 🌊 **Lenis** - The physics. Silky smooth, buttery scroll interpolation that makes navigating the site feel like gliding.
- 🧊 **React Tilt** - The tactile feel. Parallax hover effects that give flat 2D components a 3D, interactive depth.

## 🚀 Quick Start (No headaches)

Want to run this locally? It's as easy as 1, 2, 3.

```bash
# 1. Clone the repository (or download it)
git clone <repository-url>

# 2. Install the magic (dependencies)
npm install

# 3. Ignite the engine
npm run dev
```

> **Note:** Open `http://localhost:5173` in your browser. Prepare to be amazed.

## 👽 Unique Things Inside

1. **The Scroll is a Lie:** We use **Lenis** to hijack the browser's native scroll and replace it with a momentum-based, physics-driven scroll experience. Try flicking your scroll wheel!
2. **Interactive 3D Elements in 2D Space:** Hover over project cards to see **React Tilt** in action. Elements react to your mouse position, giving the illusion of depth without needing WebGL.
3. **Cinematic Reveals:** Thanks to **GSAP (GreenSock)**, elements don't just appear; they perform. Scroll down to watch sections organically reveal themselves.
